# Building
Compiled and tested on MSVC 2019.

This code requires `<Windows.h>` header file! --> Windows only.

Run those commands in a CMD console inside the source folder:
```bat 
mkdir build
cd build
cmake .. -G "Visual Studio 16 2019"
```
